# Joomla Default Title Plugin

A simple Joomla Content Plugin that automatically fills the **Article Title field** with a predefined default value when creating a new article.

This plugin was developed as part of the **Joomla Google Summer of Code (GSoC) Test Task**.

---

## Features

- Automatically inserts a **default title** when creating a new Joomla article
- Prevents empty titles during article creation
- Lightweight and easy to install
- Compatible with Joomla 5 / Joomla 6

---

### Files Description

| File | Description |
|-----|-------------|
| `defaulttitle.php` | Main plugin logic |
| `defaulttitle.xml` | Joomla plugin manifest file |
| `script.js` | Adds the default title dynamically in the article editor |

---

## Installation

1. Download the plugin ZIP file from GitHub.

2. Go to Joomla Administrator panel
